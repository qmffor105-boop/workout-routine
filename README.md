# 🏋️ 홈트레이닝 루틴 (Workout Routine)

맞춤형 홈트레이닝 운동 프로그램을 제공하는 싱글 페이지 웹앱입니다. 별도의 빌드 과정 없이 `index.html` 하나로 동작합니다.

## 🌐 바로 보기

👉 **https://qmffor105-boop.github.io/workout-routine/**

## ✨ 주요 기능

- **3가지 목표별 루틴**
  - 💪 체력향상 — 균형 잡힌 체력
  - 🔥 벌크업 — 근력 & 근비대
  - 🏃 다이어트 — HIIT & 체지방 감량
- **4주 프로그레시브 프로그램** — 주차별로 강도가 점진적으로 증가하며 마지막 주는 디로드(회복) 주간
- **요일별 운동 카드** — 카디오, 근력, 코어, 플라이오메트릭, 스트레칭 등 카테고리별 뱃지 표시
- **운동 체크 기능** — 완료한 운동을 체크해 진행률을 확인
- **인터벌 타이머** — 운동/휴식 시간 측정을 위한 내장 타이머
- **루틴 셔플** — 같은 목표 내에서 운동 순서를 섞어 지루함 방지
- **주차별 코칭 팁** — 각 주차마다 컨디션 관리, 영양, 수면에 관한 조언 제공

## 🚀 로컬에서 실행

```bash
git clone https://github.com/qmffor105-boop/workout-routine.git
cd workout-routine
# 브라우저로 index.html 열기
start index.html   # Windows
open index.html    # macOS
xdg-open index.html # Linux
```
